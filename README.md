
# Logpy

![Tests](https://github.com/Los-had/logpy/actions/workflows/linter.yml/badge.svg)

Simple and versatile logging library for python 3.6 above.

## download
install using pip:

```
pip install "git+https://github.com/Los-had/logpy.git#egg=logpy"
```
